# LALOSANkill — 美术学习助教：完整构建方案

> 目标：把LALOSAN十年的教学能力封装成一个 AI 技能，学生可以随时获得精准、个性化的绘画学习指导。
> 它不是录播课，不是知识库，而是一个**能看画、能诊断、能开方、能跟进**的 AI 助教。

## 一、这是什么

### 用一句话说

学生把画发给 LALOSANkill → LALOSANkill 用LALOSAN的教学方法诊断问题 → 给出一个具体的改正方法 + 一个当天能做完的练习 → 下次学生再发，LALOSANkill 知道上次说了什么，继续推进。

### 和现有东西的区别

| | 录播课 | 199 元知识库 | 微信跟LALOSAN聊 | **LALOSANkill** |
|---|---|---|---|---|
| 个性化 | ❌ 所有人看一样的 | ❌ 自取，不诊断 | ✅ 但LALOSAN肉身有限 | ✅ 1 对 1，无限复用 |
| 能看画诊断 | ❌ | ❌ | ✅ | ✅ |
| 能跟进进度 | ❌ | ❌ | ❌ 靠记忆 | ✅ 自动记录 |
| 24 小时可用 | ✅ | ✅ | ❌ | ✅ |
| 会进化 | ❌ | ❌ | ❌ | ✅ 每次互动都在积累数据 |

### 学生视角的一天

```
学生："老师，我画了一张静物，总觉得哪里不对。"

（上传照片：一张苹果和陶罐的素描，整体偏灰，罐子口画歪了）

LALOSANkill：
"我看到了两个问题，按优先级说——

1️⃣ 罐子口是斜的。不是你的手不稳，是你起形的时候直接画了罐子的细节形状，
没先确定它的整体外轮廓。你先用「外轮廓观察法」的眯眼法，把罐子当成一个
剪影来看，画准剪影再往里收。

2️⃣ 画面发灰，因为你的暗部不敢给。明暗交界线以下，怀疑自己一次给到位。
记住一句：暗部一次给足，亮部慢慢收。

今天的练习（15 分钟）：
用眯眼法重新画这个罐子的外轮廓剪影，3 张，每张 2 分钟。
画完发给我，我看你的外轮廓有没有比这张更准。"

—— 语气、方法、标准，全部来自LALOSAN。
```

```
├─────────────────────────────────────────┤

├─────────────────────────────────────────┤

```

### 组件 1：LALOSAN人格层

定义 LALOSANkill "说话像LALOSAN"的所有规则。从录音和教学实录中提取。

```markdown
## 人格定义

- 称呼学生为"你"，不叫"同学""亲"
- 指出问题直接、不绕弯，但结尾一定给出路（"这个问题有办法"）
- 常用句式：
  - "你看，这个地方其实就一个问题——……"
  - "不是你的手的问题。是你的眼睛还没……"
  - "我教了十年，发现学生卡在这里基本都是因为……"
  - "今天你就做一件事：……"
- 必杀技：把学生觉得玄乎的问题翻译成一个可操作的步骤
- 禁止：说"多练练就好了"——必须给出具体练什么、练多久、练到什么标准算过
```

### 组件 2：诊断引擎

这是 LALOSANkill 的核心。输入 = 学生画作 + 自述问题，输出 = 按优先级排列的根因列表。

诊断逻辑 = 症状 → 锁定知识域 → 排查该域常见错误 → 确认根因 → 匹配技法。

LALOSAN现有的诊断框架（从录音和教学实录中提取）：

| 学生说 | 症状 | 可能根因 | 对应技法 |
|--------|------|---------|---------|
| "形不准" | 比例跑、重心歪、特征不准 | ①没看整体就画细节 ②画的是概念不是看到的 | 外轮廓观察法 / 负空间法 |
| "画面脏/灰" | 调色过多、暗部反复涂抹 | ①暗部不敢给 ②补色调色比例错 | 明暗交界线穿插法 / 五色层次 |
| "画不完" | 卡在某个阶段反复改 | ①没有分步骤标准 ②完美主义中断 | 五步绘画法 |
| "不知道画什么" | 空白画布恐惧 | ①素材库空 ②没有创作决策路径 | 创作决策树 / 每日修行 |
| "坚持不住" | 三天打鱼 | ①没有可见进步 ②练习没有标准 | 每日修行打卡 / 进步对照表 |
| "线条碎" | 短线条反复描 | ①不敢一笔画完 ②眼睛没跟上手的节奏 | 外轮廓法 / 速写一笔到底 |
| "体积感弱" | 画面扁平 | ①没有明暗交界线意识 ②黑白灰层次不分 | 横五竖五塑造法 |

诊断优先级 = 基础问题 > 进阶问题。永远先解决"看不准"，再解决"画不好"。

### 组件 3：技法处方库

每个技法按统一格式存储，确保 LALOSANkill 能精准检索和输出。

```markdown
## 技法卡片：[名称]

- **解决的问题**：（一句话）
- **前置条件**：（学生需要先掌握什么？无则写"零基础可学"）
- **步骤**：（3-5 步，每步有执行标准）
- **常见错误**：（2-3 个，含原因和修正方法）
- **练习处方**：（具体练什么 + 练多久 + 什么标准算过）
- **来自LALOSAN的哪句话**：（从录音/教学实录中提取的金句或原话）
- **下一个技法**：（学完这个后推荐学什么）
```

### 组件 4：学习路径图

技法之间不是平的，有先后顺序。这个路径图决定了 LALOSANkill 给学生推荐练习时不乱推荐。

```
阶段 0：零基础入口
  └─ 外轮廓观察法 → 负空间法 → 30秒速写

阶段 1：起形能力建立
  └─ 五步绘画法 Step 1-2 → 横五竖五塑造法 → 明暗交界线

阶段 2：完整画面能力
  └─ 五步绘画法 Step 3-5 → 布褶万能公式 → 速写体系

阶段 3：风格与创意
  └─ 大师思维解码 → 创作决策树 → 色彩情感应用

阶段 4：应试/专业
  └─ 考前策略 → 押题无效论 → 盯人管理法
```

每个学生的初始定位 = 看一张画 → 判断当前在哪个阶段 → 从该阶段的第一个缺口开始补。

## 三、你需要准备什么（三件事）

### 第一件：转录所有教学录音

这是最重要的一步。你的录音里藏着你教学能力的 DNA：

- 你怎么给学生纠错（诊断句式）
- 你用哪些比喻和例子（教学语言）
- 你对不同年龄/水平学生说的话有什么不同（适配策略）
- 你在什么情况下说哪句话（触发条件）

**具体做法：**

```bash
# 1. 把所有录音文件放到一个文件夹
# 2. 用 faster-whisper 批量转录（你已经有这个工具链）
# 3. 转录后让 Claude Code 做标注：
#    - 标记每一段"纠正时刻"（学生有问题 → LALOSAN回应）
#    - 提取句式模板（"你看，这个地方其实就是……"）
#    - 提取技法口诀（"暗部一次给足，亮部慢慢收"）
```

转录后的加工命令（我可以直接帮你做）：

> "把这段教学录音转录文本，提取出：①每个纠错时刻（标注学生问题→LALOSAN诊断→LALOSAN处方）②LALOSAN的口头禅和金句 ③按技法分类整理所有纠正案例"

### 第二件：把 IMA 知识库喂进 wiki

IMA 知识库里的教学文件，如果有 wiki 里还没覆盖的内容，逐篇 ingest 进 wiki。重点是：

- 教学案例（某某学生，什么问题，怎么解决的）
- 你独创的技法或口诀（wiki 里可能还没写）
- 各年龄段/目标的差异化教法

已经覆盖的不用重复。查漏补缺即可。

### 第三件：构建"症状→技法"映射表

这是诊断引擎的燃料。上面的表只列了最核心的7条。你需要从十年经验里把所有链路补全。

**我会怎么帮你做：** 你把录音转录文本给我，我逐条提取"LALOSAN看到了什么问题 → 判断是什么根因 → 用了哪个技法来解决"，自动生成完整的映射表。

## 五、进化机制

LALOSANkill 不是写完就死的。它和学生互动的过程，就是它进化的燃料。

### 进化回路

```
学生提交画作
    ↓
LALOSANkill 诊断 → 给出处方
    ↓
学生练习后再次提交
    ↓
LALOSANkill 判断：处方有效/无效？
    ↓
┌─ 有效：记录"症状A + 技法B = 有效"，权重+1
│
└─ 无效：标记待审核，进入"疑难病例库"
    ↓
    LALOSAN定期（每周一次）看疑难病例
    ↓
    LALOSAN给出新的诊断逻辑 → 更新映射表
    ↓
    LALOSANkill 学会了一个新情况
```

### 三个进化维度

| 维度 | 怎么进化 | 频率 |
|------|---------|------|
| 诊断越来越准 | "症状→技法"映射从 30 条涨到 100+ 条 | 每次新录音消化 |
| 语气越来越像LALOSAN | 每次成功诊断的句式被记录和复用 | 每次LALOSAN审核 |
| 覆盖面越来越广 | 从素描速写扩展到色彩、创作、考前 | 每季度新增一个知识域 |

### LALOSAN每周只需做一件事

打开"疑难病例库"（我帮你整理的），看那些 LALOSANkill 没诊断准的案例，告诉我："这个学生真正的问题不是 A，是 B。"我更新映射表。**每周 30 分钟。**
