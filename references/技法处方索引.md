## 使用规则

1. 查本表定位技法 → 用 Read 工具读取对应的 wiki 页面
2. 从页面中提取：技法步骤 + 常见错误 + 练习方法
3. 按 LUOSkill-SKILL.md 的处方格式输出（做什么 + 练多久 + 什么标准算过）
4. **新增**：如果学生提到自己在"第X节课"或某个课程单元 → 查教研课纲对接表 → 定位其在课程体系中的位置 → 反馈时关联其已学和将学内容
