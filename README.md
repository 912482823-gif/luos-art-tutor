# LALOSANkill — 美术学习助教

你画一张画，发给我。我帮你看哪里有问题、怎么改、练什么。

我不是帮你画画的 AI。我是帮你**学会画画**的 AI 助教。

## 怎么用

1. 把整个文件夹下载到你的电脑
2. 安装 [Claude Code](https://claude.ai/code)
3. 在 Claude Code 里加载本 Skill：把 `LALOSANkill-SKILL.md` 拖入对话，或者用 `/skill` 命令加载
4. 发一张你画的画给我，我会先了解你的情况，然后帮你诊断

## 我能做什么

- 看你的画，指出问题（素描/速写/色彩/装饰画都行）
- 告诉你问题背后的真实原因（不是你手笨，是你不知道看哪里）
- 给你一个当天就能做的练习（具体到练多久、什么标准算过）
- 记住你的学习进度，每次点评都会参考你之前的卡点
- 帮你判断一个老师或机构适不适合你（横纵分析法）

## 我不能做什么

- 帮你画——我是助教，不是代笔
- 评价你的画"美不美"——我只能讲技术，不能替代审美判断
- 雕塑、装置、数字媒体——我的领域是平面绘画
- 考研/考级打分——需要具体院校的评分标准

## 适合谁

任何在学画画的人——小学生、中学生、大学生、成年人、退休了想找个事做的。

不管你学了三年还是零基础刚摸笔，我都能给你当下最有用的反馈。

## 谁教的

教了十年美术，服务过 1000+ 家庭。

我的核心主张：**美术教学不是教"怎么画"，而是教"怎么看"和"怎么想"。** 天赋和感觉可以转化为可量化、可执行、可复制的科学能力。

这个 Skill 是我的 AI 分身——用我的诊断方法、我的教学语言、我的技法体系来帮你看画。
